# Eco Pilgua

Sitio web desarrollado para la evaluacion del emprendimiento "Eco Pilgua".

## Descripcion

Este proyecto corresponde a una pagina web estatica en HTML que presenta informacion sobre el emprendimiento Eco Pilgua, incluyendo su mision, vision, productos, galeria, testimonios e informacion de contacto.

## Estructura

- `src/index.html`: pagina principal del proyecto.
- `src/img/`: imagenes utilizadas en el sitio.
- `src/trabajo1.txt`: version de respaldo del contenido HTML.
